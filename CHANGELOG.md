## [Unreleased]

### Added
- Initial Go API scaffold.
- `GET /health` liveness/readiness endpoint with build version and commit.
- `POST /v1/auth` HTTP Basic Auth validated via LDAP bind.
- `GET /v1/user/:uid` LDAP attribute lookup.
- LDAPS client with CA cert loading, TLS 1.2+, and filter escaping.
- Gateway vs standalone authentication modes.
- Structured request logging via `log/slog`.
- OpenAPI 2.0 spec generated by swaggo, served at `/swagger/*` when enabled.
- Unit tests for handlers and config loader.
- Multi-stage `Dockerfile` with native cross-compilation via `$BUILDPLATFORM`,
  OpenShift-compatible runtime (UID 1001, group 0, `HOME=/tmp`), and `apk upgrade`
  for CVE hygiene. `.dockerignore` keeps the build context minimal.
